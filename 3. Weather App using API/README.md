# 🌦️ Weather App using API (CLI + GUI)

Real-time weather app using OpenWeather API.

---

## 🚀 Features
✅ Live weather data
✅ API integration
✅ CLI + GUI versions
✅ Temperature + Humidity + Condition
✅ Beginner friendly
✅ Clean commented code

---

## 🔑 Setup API Key
1. Create account on openweathermap.org
2. Copy API key
3. Paste inside config.py

---

## 🛠 Install
pip install -r requirements.txt

---

## ▶ Run CLI
python cli_weather.py

---

## ▶ Run GUI
python gui_weather.py

---

## 📚 Concepts Used
- API calls
- JSON parsing
- requests library
- Tkinter GUI
- Error handling
